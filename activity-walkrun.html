ABOUT

GPSウォーク&ランとは

GPSウォーク&ランは、スマートフォンのGPSアプリを使って、 歩いたり走ったりした軌跡で地図上に絵や文字を描くイベントです。 タイムも順位も競わない、GPSランナーが提唱する「新しいランニング文化」の体験版。

GPSランナー志水直樹が事前にルートを設計し、当日は参加者と一緒に歩きます。 1人で描くソロ作品から、数百人で分担して描く大規模イベントまで対応可能。 完成した作品はその場で全員とシェアし、SNS投稿で地域の話題にもなります。

TYPES

2つのスタイル

WALK

GPSアートウォーク

歩いてゆっくりアートを描くスタイル。お散歩感覚で参加でき、地域の新しい魅力も発見。

  • 所要時間：1〜2時間
  • 距離：3〜4km
  • 対象：全年齢（お子さま・シニアも歓迎）
  • 難易度：やさしい
RUN

GPSアートラン

走って壮大なアートを描くスタイル。ランニング愛好家向けの本格コース。

  • 所要時間：1〜6時間
  • 距離：5〜42km
  • 対象：ランナー向け
  • 難易度：中〜高

FEATURES

イベントの魅力

🚶

誰でも参加OK

歩くだけでアートが完成。運動が苦手な方、お子さま連れ、車いすの方も参加できるルート設計が可能です。

🗺️

まちの再発見

普段通らない路地や公園を巡るルート設計。地元の人でも知らなかったスポットに出会えます。

📱

SNS映えする成果

完成したGPSアートは地図上で見ると圧巻。参加者がSNSでシェアすることで、地域のPRにもつながります。
